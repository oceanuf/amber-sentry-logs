## 🔍 穿透算法说明

### 核心逻辑
1. **数据获取**: 使用 `pro.fund_portfolio` 获取最近4个季度的持仓数据
2. **权重计算**: 基于基金持仓市值计算股票在基金中的权重
3. **价值映射**: 将基金投资金额按权重映射到底层股票
4. **数据聚合**: 合并同一股票在不同基金中的持仓

### 计算公式
```
股票在基金中的权重 = 股票持仓市值 / 基金总持仓市值
股票在投资中的价值 = 基金投资金额 × 股票在基金中的权重
股票占总资产比例 = 股票在投资中的价值 / 总投资金额 × 100%
```
