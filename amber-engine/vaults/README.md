# 🏛️ 档案馆核心 - Vaults目录

## 📋 目录规范
基于[2614-001]档案馆单体仓库同步法典 V3.1

### 允许文件类型
- **仅限 `.md` 内容文件**
- 按模组归类存放
- 严禁存放二进制文件、敏感数据、临时文件

### 模组分类
1. **Assets/** - 资产档案
2. **Formulas/** - 公式算法
3. **Protocols/** - 协议规范
4. **Reports/** - 专项报告
5. **Templates/** - 模板文件

## 🚫 禁止内容
- 真实资金数据
- 个人隐私信息
- 硬编码Token
- 绝对路径引用
- 非Markdown格式文件

## 📁 当前结构
```
vaults/
├── README.md (本文件)
├── Assets/      # 资产档案目录
├── Formulas/    # 公式算法目录  
├── Protocols/   # 协议规范目录
├── Reports/     # 专项报告目录
└── Templates/   # 模板文件目录
```

## 🔒 安全要求
所有文件必须通过 `scripts/github/sync_clean.sh` 安全审查后才能同步至GitHub。
