```json
"models": {
  "mode": "merge",
  "default": "deepseek-reasoner", // 新增：强制默认使用推理模型
  "providers": {
    "deepseek": {
      "models": [{
        "id": "deepseek-reasoner",
        "reasoning": true,
        "contextWindow": 128000,
        "options": { "temperature": 0.1 } // 新增：降低随机性，确保财经数据严谨
      }]
    }
  }
}
```

### 2. 代理配置优化

    }
  }
}
