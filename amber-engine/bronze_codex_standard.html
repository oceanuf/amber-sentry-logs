核心数学公式

总评分计算公式
\[ \text{Final Score} = 0.50 \times S_{\alpha} + 0.30 \times S_{strategy} + 0.15 \times S_{robust} + 0.05 \times S_{efficiency} \]

其中 \(S_{\alpha}\)、\(S_{strategy}\)、\(S_{robust}\)、\(S_{efficiency}\) 分别代表四大维度的得分，权重系数永久锁定。

Alpha Ladder阶梯函数
def calculate_alpha_score(alpha_val):
if alpha_val >= 8.0: return 100 # S+ 极佳
elif 5.0 <= alpha_val < 8.0: return 95 # S 卓越
elif 2.0 <= alpha_val < 5.0: return 85 # A 优秀
elif 0.0 <= alpha_val < 2.0: return 65 # B 合格 (生死线)
elif -2.0 <= alpha_val < 0.0: return 45 # C 弱势
elif -5.0 <= alpha_val < -2.0: return 25 # D 危险
else: return 5 # E 崩溃 (惩罚分)

60分生死线制度: 动态超额收益跑输大盘0%即视为不及格(45分)，确保算法对弱势基金的严格筛选。
