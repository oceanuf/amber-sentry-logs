# 🛠️ 任务指令 [2613-090-A]：数据抓取引擎实装

## 目标
实现 `scripts/fetch_bronze_data.py`，完成 50 支标的的 30 日历史数据初始化。

## 执行要求
1. **数据源**：调用 Tushare Pro 的 `fund_nav` 接口。
2. **逻辑内核**：
   - 抓取 T-31 至 T-1 的单位净值（unit_nav）。
   - 实现 T+1 判定：每日 08:00 后才抓取前一交易日数据。
   - 容错：若 T-1 缺失，自动回退至 T-2 并标注。
3. **输出格式**：
   - 路径：`data/nav_history/{code}.json`
   - 格式：`[{"date": "20260325", "nav": 1.234, "change": 0.01}, ...]`
4. **频率控制**：每分钟抓取不超过 10 支，防止 API 熔断。

## 验收标准
- `data/nav_history/` 下生成 50 个非空 JSON 文件。
- 运行日志无 `Auth Error` 或 `Rate Limit` 报错。
