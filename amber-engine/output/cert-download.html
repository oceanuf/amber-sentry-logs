解决 https://finance.cheese.ai 访问安全警告

🔐 问题描述

访问 https://finance.cheese.ai 时出现"您的连接不是私密连接"警告。

🎯 解决方案

方法1：一键安装（推荐）

  1. 下载：install-certificates.bat
  2. 右键点击 → 以管理员身份运行
  3. 按照提示完成安装
  4. 重启所有浏览器窗口
下载一键安装工具

方法2：手动安装

步骤1：安装根CA证书

  1. 下载：根CA证书
  2. 双击证书文件
  3. 点击"安装证书"
  4. 选择"本地计算机" → 下一步
  5. 选择"将所有的证书都放入下列存储"
  6. 点击"浏览" → 选择"受信任的根证书颁发机构"
  7. 完成安装

步骤2：安装服务器证书

  1. 下载：服务器证书
  2. 双击证书文件
  3. 点击"安装证书"
  4. 选择"本地计算机" → 下一步
  5. 选择"将所有的证书都放入下列存储"
  6. 点击"浏览" → 选择"中间证书颁发机构"
  7. 完成安装

🔄 安装后步骤

  1. 清除浏览器缓存：
    • Chrome: 访问 chrome://net-internals/#hsts → 清除HSTS
    • Edge: 访问 edge://net-internals/#hsts → 清除HSTS
    • Firefox: 设置 → 隐私与安全 → 清除数据
  2. 重启浏览器：完全关闭所有浏览器窗口后重新打开
  3. 验证安装： 
    certutil -store Root | findstr "Cheese Intelligence"

⚠️ 故障排除

如果仍有警告：

  1. 重启电脑：确保系统完全加载新证书
  2. 检查系统时间：确保电脑时间正确
  3. 尝试不同浏览器：Chrome/Edge/Firefox
  4. 清除SSL状态： 
    certutil -urlcache * delete
